# ACode

Aidan's custom cross-platform code editor, built on Tauri 2 + React + CodeMirror 6.

**Stack:** Tauri 2 · React 19 · TypeScript · Vite · CodeMirror 6 · Zustand · Claude Code SDK

## Docs

- [Architecture](docs/architecture.md) — stack decisions, directory structure, IPC patterns
- [Contributing](docs/contributing.md) — setup, dev workflow, conventions

## Quick Start

```bash
npm install
npm run tauri dev
```

> Requires Rust + OS-specific Tauri dependencies. See [contributing.md](docs/contributing.md).
